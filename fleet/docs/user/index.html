Fleet is Frappe app for fleet management. It is to be used along with ERPNext. Maintainence Schedule and Support module from ERPNext can be used for tracking Issues, Maintainence Visits and Schedule. Purchase Invoices for fuel and expenses can also be handled by ERPNext. This App cannot be used separately and only be used along with ERPNext.

Fleet creates a desk icon for the module. In this module, there are three types of Documents viz. Fill Up Record for recording fuel fill ups, Trip Record for recording trips and Vehicle master for tracking vehicles in your Fleet. Under Setup section of this module there are Fleet Settings, in here Distance Unit of Measure can be set to Mile or KM (Currently Mile/KM is only Company / Database Wide).
